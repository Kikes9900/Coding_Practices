Semantic HTML

Sections Of A Webpage
Tag Name Description
<header> header A container for introductory content or navigation links
<nav> nav Contains a set of navigation links
<main> main Used for containing the main content of a document
<footer> footer Is used for containing end content of a document
Main Content Section Tags
Tag Name Description
<section> section Defines a section in a document
<article> article Specifies independent, self-contained content
<aside> aside Defines some content aside from the content it is placed in
<figure> figure Specifies self-contained content, like illustrations, diagrams, photos, code listings and more
<figcaption> figcaption Defines a caption fore the figure element
Semantic Media HTML
Tag Name Description
<video> video Is used to embed video content into the document
<embed> embed defines a container for an external resource, such as webpage, picture, or media player
<audio> audio Is used to embed sound content in a document
